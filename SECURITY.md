# Security Policy

## Supported Versions

This project is pre-1.0 and under active development.
Security support will evolve as the project matures.

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please report it privately.

For now:
- Open a GitHub issue and clearly mark it as a **security concern**, **or**
- If the repository provides a maintainer contact method, use that for private disclosure.

Please include:
- a description of the vulnerability
- steps to reproduce (if possible)
- impact assessment
- affected versions/commits

We will acknowledge reports and investigate as promptly as possible.
